CoDecoder By Karen Shekyan, Period 9

Info

This project is a website that teaches the user about various different ciphers and their histories and allows the user to...use the ciphers. A home page introduces the user to the topic and each of 3 pages goes more in depth about the Vigenere cipher, the RSA cipher, and Morse code. The sites use python programs to perform the encryption and decryption tasks via math and dictionaries. Each page describe how to use it. To encrypt/decrypt on the Vigenere page, enter a phrase and a keyword press 'Encode' or 'Decode' buttons. On the Morse page, type a phrase into the plaintext box to encode or a Morse-ified message into the ciphertext box to decode. On the RSA page, input 3 numbers to decrypt. More information on these processes is provided on each page. Link to the project website is in the navbar!

Files

  • final.html: me! (this file)
  • CipherSite: project home page. Serves as an introduction to the info in the project.
  • Cipherstyle.css: formatting and styles for all the html files.
  • Morse.html: page containing information about Morse code and an encryption/decryption program for the user.
  • RSA.html: page containing information about the RSA cipher and a decryption program for the user.
  • Vigenere.html: page containing information about the Vigenere cipher and an encryption/decryption program for the user.
  • DMorse.py: python program for decrypting Morse.
  • DVigenere.py: python program for decrypting Vigenere.
  • EMorse.py: python program for encrypting Morse.
  • EVigenere.py: python program for encrypting Vigenere.
  • RSA.py: python program for decrypting RSA.
  • VigenereMath.jpg: Frankensteined image displayed on the Vigenere cipher page to visualize the concept.

"Features" (Bugs)

  • Entering a non-letter character into the 'keyword' box on the Vigenere cipher page in the encode section will produce a server error. The page explains the keyword must only contain letters to avoid this.
  • Entering anything but correctly formatted Morse code into the 'Ciphertext' box on the Morse code page will produce a server error.
